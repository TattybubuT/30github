CSS font-style

Свойство font-style в CSS отвечает за стиль шрифта, позволяя делать текст курсивным, обычным или наклонным.

Обычный текст (без курсива). (font-style: normal)

Курсивный текст (если шрифт поддерживает курсив).(font-style: italic)

Наклонный текст.
Браузер наклоняет обычный шрифт, если нет курсива. (font-style: oblique)

Свойство font-weight

Свойство font-weight определяет насыщенность (жирность) шрифта, делая текст тоньше или толще.

Обычная жирность шрифта (по умолчанию).

Жирное начертание текста.

Делаает текст жирнее по сравнению с родительским элементом.

Делаает текст тоньше по сравнению с родительским элементом.

Свойство text-decoration

Свойство text-decoration добавляет различные линии к тексту, такие как подчёркивание, надчёркивание и перечёркивание.

Подчёркивание (underline)

Надчёркивание (overline)

Перечёркивание (line-through)

Отмена декора (none)

Свойство letter-spasing

Свойство letter-spacing регулирует расстояние между буквами в тексте. Оно позволяет увеличивать или уменьшать интервал между символами.

Увеличение расстояния (10px)

Расстояние в em (0.3em)

Отрицательное значение (-2px)

Свойство line-height

Свойство line-height регулирует межстрочный интервал (расстояние между базовыми линиями соседних строк). Оно играет важную роль в читабельности текста и его визуальном оформлении.

Свойство text-align

Свойство text-align в CSS используется для выравнивания строчного содержимого внутри родительского элемента. Оно позволяет управлять расположением текста, изображений и других строчных элементов по горизонтали в пределах контейнера.

Работа цветами в CSS

Красный - rgb(255, 0, 0)

Зелёный - rgb(0, 255, 0)

Синий - rgba(0, 0, 255, 0.8) (RGBA)

Жёлтый - rgba(255, 255, 0, 0.8) (RGBA)

Голубой - #00FFFF (HEX)

Пурпурный - #FF00FF (HEX)

Красный - hsl(0, 100%, 50%) (HSL)

Серый - hsl(0, 0%, 50%) (HSL)

Красный - hsla(0, 100%, 50%, 0.8) (HSLA)

Зелёный - hsla(120, 100%, 50%, 0.8) (HSLA)
